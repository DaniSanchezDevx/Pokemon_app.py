# Pokemon_app.py
Pokémon Visualizador – Un Proyecto Python para Entusiastas de Pokémon -- Este proyecto es una aplicación interactiva escrita en Python que te permite explorar la vasta información de la franquicia Pokémon. Diseñado para principiantes, ofrece una forma sencilla de buscar y visualizar datos de Pokémon desde la API de Pokedex.
