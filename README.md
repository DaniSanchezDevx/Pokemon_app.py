# Pokémon Visualizador - Explora la Franquicia Pokémon 


## Descripción

Este proyecto es una aplicación interactiva escrita en Python que te permite explorar la vasta información de la franquicia Pokémon. Diseñado para principiantes, ofrece una forma sencilla de buscar y visualizar datos de Pokémon desde la API de Pokedex.

## Funcionalidades

*   **Buscar Pokémon por Nombre:** Ingresa el nombre de un Pokémon y la aplicación recuperará su información, incluyendo:
    *   Nombre
    *   Imagen
    *   Tipo(s)
    *   Altura
    *   Peso
    *   Estadísticas Básicas (CP, Ataque, Defensa, etc.)
    *   Movimientos
*   **Buscar Pokémon por Tipo:** Explora la diversidad de tipos Pokémon buscando por tipo (e.g., “Fuego”, “Agua”, “Planta”).
*   **Visualización Interactiva:** Presenta la información en un formato claro y fácil de entender, con imágenes de alta calidad y estadísticas bien organizadas.
*   **Experiencia de Aprendizaje:** Ideal para aprender sobre la franquicia Pokémon y comprender cómo funcionan las APIs y el control de versiones.

## Tecnologías Utilizadas

*   **Lenguaje:** Python 3.x
*   **Biblioteca:** `requests` (para hacer peticiones a la API)
*   **Biblioteca:** `json` (para manejar datos JSON)
*   **Interfaz Gráfica:** `tkinter`

## Instalación y Ejecución

1.  **Clona el repositorio:**  Usa el comando `git clone [URL de tu repositorio]`
2.  **Navega al directorio del proyecto:**  Usa el comando `cd pokemon_app`
3.  **Instala las dependencias:**  Ejecuta el comando `pip install requests`
4.  **Ejecuta el programa:**  Ejecuta el comando `python pokemon_app.py`

## Cómo Usarlo

1.  El programa te pedirá que ingreses el nombre de un Pokémon.
2.  Ingresa el nombre y presiona Enter.
3.  La información del Pokémon se mostrará en la consola.
